# MinionOS

Development of a simple OS for a BeagleBoard C4 with a ARMv7 Cortex A8-Processor (OMAP 3530). 

## Following parts should be implemented:
* GPIO (ok)
* Interrupts (ok)
* Timers/System Timer (ok)
* System Calls (in work)
* Scheduler
* UART
* Filesystem (in work)
* Processes with IPC
* VM
* Basic User Apps
